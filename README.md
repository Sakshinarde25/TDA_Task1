# Personal Introduction Program

## Project Overview
This project introduces Python basics such as variables, input/output,
and string formatting.

## Objectives
- Learn Python syntax
- Use input() and print()
- Build a simple interactive program

## Setup Instructions
1. Install Python
2. Clone the repository
3. Run `python personal_intro.py`

## What I Learned
- Variables and data types
- User input handling
- f-string formatting
